# Student Performance Intelligence System

## Objective
Analyze student academic data to identify performance patterns
and students at risk of poor academic outcomes.

## Business Context
Early identification of struggling students helps institutions
provide timely interventions and improve overall results.

## Tasks
- Load and inspect the dataset
- Clean and preprocess data
- Perform exploratory data analysis (EDA)
- Visualize key trends
- Generate actionable insights

## Tools
Python, Pandas, NumPy, Matplotlib, Seaborn

# Student Performance Intelligence System

## Objective
Analyze student academic data to identify performance patterns
and students at risk of poor academic outcomes.

## Business Problem
Educational institutions often fail to identify struggling students early,
leading to poor results and higher dropout rates.

## Tasks
- Load and clean student data
- Handle missing values
- Perform exploratory data analysis (EDA)
- Visualize performance trends
- Generate insights and recommendations

## Tools
Python, Pandas, NumPy, Matplotlib, Seaborn
